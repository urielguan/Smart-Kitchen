# 智慧食安管理平台 - WEB端前端演示

## 📋 项目说明

本项目是**智慧食安管理平台**的纯前端可交互演示版本，采用原生 HTML + CSS + JavaScript 实现，无需后端服务器，可直接在浏览器中打开运行。

---

## 🚀 快速开始

### 运行方式

1. **直接打开**：双击 `index.html` 文件，用浏览器（Chrome / Firefox / Edge）打开即可
2. **本地服务器**（可选）：
   ```bash
   # 使用 Python 启动本地服务器
   python3 -m http.server 8080
   # 或使用 Node.js
   npx http-server -p 8080
   ```
   然后访问 `http://localhost:8080`

### 外部依赖

- **Element Plus CSS**（通过 CDN 引入）：需要联网加载样式
- 如需离线使用，请下载 Element Plus CSS 文件到本地并修改 `index.html` 中的引用路径

## 🎨 交互细节

### 全局交互

- **侧边栏导航**：点击菜单项切换页面，激活项高亮蓝色，面包屑同步更新
- **Toast 提示**：操作成功/失败后右上角显示绿色/红色提示，3秒后自动消失
- **弹窗**：
  - 打开时有 slideDown 动画
  - 点击遮罩层空白处关闭
  - 点击右上角 ✕ 关闭
- **删除确认**：点击删除按钮弹出浏览器原生 confirm 确认框
- **审核弹窗**：包含审核意见文本域，通过/驳回两个按钮

### 搜索过滤

- 所有搜索均为**前端过滤 Mock 数据**（无后端接口）
- 支持多条件组合搜索
- 点击"重置"按钮恢复全部数据

### AI 建议功能

- **物料管理**：点击"AI建议"按钮后，自动填入临期提醒天数=7天、预警期=30天，并在按钮下方显示悬浮提示框（4秒后自动消失）
- **菜谱管理**：
  - 烹饪时长 AI 建议：自动填入45分钟
  - 目标温度 AI 建议：自动填入95℃

### 动态表格

- **入库单/出库单**：物料明细表格支持动态增删行
- **菜谱管理**：食材列表支持动态增删行

## 📊 数据说明

### Mock 数据存储

所有数据存储在 `js/mock-data.js` 的全局对象 `mockData` 中，包含：
- `warehouses`：仓库列表（4条）
- `inboundOrders`：入库单（3条）
- `outboundOrders`：出库单（3条）
- `stocktakeOrders`：盘点单（3条）
- `materials`：物料列表（8条）
- `recipes`：菜谱列表（6条）

### 数据持久化

- **增删改操作**：直接操作内存中的 `mockData` 对象
- **刷新页面**：数据恢复为初始值（无本地存储）
- 如需持久化，可自行扩展 `localStorage` 或对接真实后端接口

## 📝 代码规范

### 命名约定

- **CSS 类名**：kebab-case（如 `.stat-card`、`.btn-primary`）
- **JavaScript 变量**：camelCase（如 `currentPage`、`editingWarehouseId`）
- **JavaScript 函数**：camelCase（如 `renderSidebar()`、`showToast()`）

### 注释规范

- **CSS**：每个模块用注释分隔（如 `/* ========== 侧边栏 ========== */`）
- **JavaScript**：
  - 文件头部说明文件用途
  - 每个函数前有简要说明
  - 关键逻辑行内注释

## 🔧 扩展开发建议

### 对接真实后端

1. 将 `js/mock-data.js` 中的数据替换为 API 请求
2. 在 `utils.js` 中封装统一的 `fetch` 或 `axios` 请求函数
3. 修改各模块的增删改查函数，调用真实接口

### 添加新模块

1. 在 `js/` 目录下新建模块文件（如 `supplier.js`）
2. 在 `sidebar.js` 的 `renderPage()` 函数中添加分支
3. 在 `index.html` 中引入新模块脚本

### 优化建议

- **分页功能**：当前仅显示总条数，可扩展为真实分页
- **表单验证**：当前仅校验必填项，可增强正则验证
- **权限控制**：根据用户角色显示/隐藏操作按钮
- **响应式布局**：当前为 PC 端设计，可适配移动端
